자주 묻는 질문

더 궁금한 점은 앱의 도움말 → 피드백 보내기, 또는 메일로 물어보세요.

무료인가요?

베타 기간 동안 무료입니다. AI 사용료는 본인의 API 키로 각 제공자(Google·Anthropic·OpenAI)에 직접 지불하는 구조라, Mind가 중간에서 웃돈을 받지 않습니다.

API 키가 꼭 필요한가요?

AI 기능(채팅·에이전트)에만 필요합니다. 보드·코드맵·에디터·협업은 키 없이 전부 사용할 수 있습니다. Google AI Studio에서 Gemini 키를 무료로 시작할 수 있어요.

VS Code 확장과 무엇이 다른가요?

Mind는 확장이 아니라 VS Code 오픈소스를 기반으로 만든 독립 앱입니다. 보드가 첫 화면이고, 아이디어와 코드의 연결이 제품의 중심입니다. 기존 VS Code와 나란히 설치해도 서로 간섭하지 않습니다.

내 코드나 보드가 서버로 전송되나요?

아니요. 파일은 내 컴퓨터에만 있습니다. AI에게 보내는 내용은 내 키로 해당 AI 제공자에게 직접 전송되고, 실시간 협업을 켠 보드만 동기화 서버를 경유합니다. 익명 사용 통계는 동의한 경우에만(opt-in) 수집되며, 코드 내용은 어떤 경우에도 수집하지 않습니다. 자세한 내용은 개인정보처리방침을 참고하세요.

어떤 AI 모델을 쓸 수 있나요?

Anthropic Claude, OpenAI GPT, Google Gemini 계열 6개 모델을 지원하며, 채팅 하단의 모델 선택기에서 언제든 전환할 수 있습니다. 키를 등록한 모델만 활성화됩니다.

다른 확장 프로그램도 설치할 수 있나요?

네. 오픈 마켓플레이스(Open VSX)가 연결되어 있어 테마, 언어팩, 각종 개발 도구 확장을 확장 탭에서 검색해 설치할 수 있습니다.

Mac이나 Linux 버전은 없나요?

현재 베타는 Windows 10/11(64-bit)만 지원합니다. Mac·Linux는 수요를 보며 검토 중입니다 — 원하신다면 앱의 피드백 기능으로 알려주세요. 우선순위를 정하는 데 큰 도움이 됩니다.

협업은 어떻게 동작하나요?

보드 좌상단 상태 칩에서 공유 코드를 복사해 전달하면, 상대가 자기 Mind에서 그 코드로 참여합니다. 변경 사항은 충돌 없이 실시간 병합되고(CRDT 방식), 연결이 끊겨도 각자 작업한 뒤 다시 만나면 합쳐집니다.

인터넷 없이도 쓸 수 있나요?

보드·코드맵·에디터는 완전히 오프라인으로 동작합니다. AI 기능과 실시간 협업, 확장 설치만 인터넷이 필요합니다.
