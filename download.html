Mind 다운로드

무료 베타 · Windows 10 / 11 (64-bit) · 약 210MB

MindSetup.exe 다운로드

관리자 권한 불필요 · 기존 VS Code와 나란히 설치 가능

설치 파일 실행

내려받은 MindSetup.exe를 실행합니다. 사용자 폴더에 설치되므로 관리자 권한을 묻지 않습니다.

SmartScreen 경고가 뜬다면

베타 기간에는 코드서명 준비 중이라 Windows가 "알 수 없는 게시자" 경고를 표시할 수 있습니다. 추가 정보를 클릭한 뒤 실행을 누르면 정상적으로 설치됩니다.

설치 완료

설치가 끝나면 Mind가 실행되고 시작 메뉴와 바탕화면에 아이콘이 생깁니다. 첫 실행에서 표시 언어를 고른 뒤, 시작 가이드를 따라 5분 만에 첫 보드를 만들어보세요.

업데이트와 제거

업데이트 (자동)

새 버전이 나오면 Mind가 자동으로 확인해 백그라운드에서 내려받고, 준비되면 "재시작하여 업데이트" 알림을 띄웁니다. 다시 다운로드하실 필요가 없습니다. 보드와 설정은 업데이트 후에도 그대로 유지됩니다.

제거

Windows 설정 → 앱 → 설치된 앱에서 Mind를 찾아 제거하면 됩니다. 내 폴더에 저장된 보드 파일(.mind 폴더)은 삭제되지 않습니다.

설치에 문제가 있나요? 메일로 알려주세요. 시스템 요구사항: Windows 10 이상 (64-bit), 여유 공간 600MB.
