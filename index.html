실시간 협업

코드 한 줄 공유하면 여러 명이 같은 보드를 동시에 편집합니다. 커서까지 실시간으로.

아이디어가 어떤 코드가 됐는지, 잊히지 않습니다

보드의 메모를 코드 파일에 연결하면, 에디터에서 그 파일을 열 때마다 "보드에서 보기" 링크가 나타납니다. 왜 이 코드를 짰는지가 항상 한 클릭 거리에 있습니다.

  • 코드맵에서 파일 구조를 그림으로 탐색
  • 보드 메모 ↔ 코드 파일 양방향 연결
  • 멀티 폴더 워크스페이스 지원

내 API 키로, 내 컴퓨터에서

Mind는 구독형 AI 중개상이 아닙니다. 쓰는 만큼만 내는 내 키(BYOK)로 동작합니다.

🔑

키는 암호화 저장

Gemini·Claude·GPT 키는 이 컴퓨터에 암호화되어 저장되고, 외부로 전송되지 않습니다.

💾

파일은 내 폴더에

보드도 코드도 전부 내 폴더의 파일입니다. 클라우드 강제 없음, 잠금(lock-in) 없음.

📊

데이터 수집은 선택

익명 사용 통계는 기본 꺼짐(opt-in). 코드 내용은 어떤 경우에도 수집하지 않습니다.

FAQ

무료인가요?

베타 기간 동안 무료입니다. AI 사용료는 본인의 API 키로 각 제공자(Google·Anthropic·OpenAI)에 직접 지불하는 구조라, Mind가 중간에서 웃돈을 받지 않습니다.

API 키가 꼭 필요한가요?

AI 기능(채팅·에이전트)에만 필요합니다. 보드·코드맵·에디터·협업은 키 없이 전부 사용할 수 있습니다. Google AI Studio에서 Gemini 키를 무료로 시작할 수 있어요.

VS Code 확장과 무엇이 다른가요?

Mind는 확장이 아니라 VS Code 오픈소스를 기반으로 만든 독립 앱입니다. 보드가 첫 화면이고, 아이디어→코드 연결이 제품의 중심입니다. 기존 VS Code와 나란히 설치해도 서로 간섭하지 않습니다.

내 코드나 보드가 서버로 전송되나요?

아니요. 파일은 내 컴퓨터에만 있습니다. AI에게 보내는 내용은 내 키로 해당 AI 제공자에게 직접 전송되고, 실시간 협업을 켠 보드만 동기화 서버를 경유합니다. 익명 사용 통계는 동의한 경우에만(opt-in) 수집됩니다.
