설치부터 첫 아이디어를 코드에 연결하기까지.

설치하고 실행하기

MindSetup.exe를 받아 실행하면 1분 안에 설치가 끝납니다. 관리자 권한이 필요 없고, 기존 VS Code와도 충돌하지 않습니다. 첫 실행에서 표시 언어(한국어/English)를 고르면 준비 완료입니다.

폴더 열고 보드 펼치기

앱을 켜면 보드가 기다리고 있습니다. 코드맵 화면의 폴더 열기 버튼으로 작업할 폴더를 열어두면 보드가 그 폴더에 자동 저장됩니다. 메모(더블클릭)·도형·화살표·펜으로 생각을 펼치고, PDF나 이미지도 끌어다 놓아보세요.

AI 키 등록하기

Ctrl+Shift+P → Mind: API 키 설정을 실행하고 사용할 AI(Gemini·Claude·GPT)의 키를 붙여넣으세요. 셋 중 하나만 있으면 되고, Gemini는 Google AI Studio에서 무료로 시작할 수 있습니다. 키는 이 컴퓨터에 암호화되어 저장됩니다.

보드와 코드 연결하기

보드에서 메모를 클릭하고 코드 파일 연결을 누르면 파일 검색으로, 코드맵에서 선택을 누르면 지도에서 직접 골라 연결할 수 있습니다. 연결된 파일을 에디터에서 열면 보드에서 보기 링크가 나타나 — 이 코드가 어떤 아이디어에서 나왔는지 언제든 되짚을 수 있습니다.

AI 에이전트에게 맡기기

오른쪽 채팅 패널에서 모델을 고르고 대화를 시작하세요. 에이전트 모드에서는 "이 메모대로 초안 파일 만들어줘" 같은 요청에 AI가 직접 파일을 만들고 수정하며, 변경 사항은 수락/거부로 검토할 수 있습니다.

함께 그리기 (선택)

보드 왼쪽 위 상태 칩을 눌러 공유 코드를 복사해 동료에게 보내면, 같은 보드를 실시간으로 함께 편집할 수 있습니다. 커서까지 서로 보입니다.
