기획부터 코드까지, 끊기지 않는 흐름

Mind의 네 가지 축을 자세히 소개합니다.

아이디어 보드

앱을 켜면 바로 보이는 무한 화이트보드입니다. 회의 내용, 기획 아이디어, 화면 스케치를 형식에 얽매이지 않고 펼쳐놓으세요.

  • 메모·텍스트·도형(사각형/원/마름모)과 화살표 연결
  • 펜과 형광펜으로 자유롭게 드로잉
  • PDF·Word·PPT 문서를 끌어다 놓으면 보드 위에 펼쳐지고, AI가 내용을 읽을 수 있습니다
  • 클립보드 이미지 바로 붙여넣기
  • 공유용 HTML·PNG·SVG 내보내기
  • 열어둔 폴더에 자동 저장 — 별도 저장 버튼이 없습니다

코드맵과 살아있는 링크

프로젝트의 파일 구조를 방사형 지도로 탐색하고, 보드의 아이디어를 실제 코드 파일과 연결하세요. 연결은 양방향으로 살아 있습니다.

  • 폴더를 클릭하며 파일 구조를 지도처럼 탐색, 파일 이름 검색
  • 보드 메모에서 파일을 연결 — 검색으로 고르거나 코드맵에서 직접 클릭
  • 연결된 파일을 에디터에서 열면 "보드에서 보기" 링크가 표시
  • 연결된 파일·폴더에는 지도 위에 링크 배지가 표시
  • 파일 생성·삭제를 실시간 반영, 새 파일·폴더 생성도 지도에서 바로

AI 에이전트

Mind는 VS Code 오픈소스를 기반으로 한 완전한 코드 에디터이고, 그 위에 AI 채팅과 에이전트가 내장되어 있습니다. 별도 구독 없이 본인의 API 키로 동작합니다.

  • Claude·GPT·Gemini 6개 모델을 채팅에서 바로 전환
  • 에이전트 모드: 파일 생성·수정·프로젝트 탐색을 AI가 직접 수행
  • 변경 사항은 수락/거부로 검토 후 반영
  • 보드에 올린 문서(PDF·Word·PPT)의 텍스트를 AI가 읽고 답변에 활용
  • API 키는 컴퓨터에 암호화 저장, 외부 전송 없음

실시간 협업

보드 왼쪽 위의 상태 칩에서 공유 코드를 복사해 전달하면, 상대방이 같은 보드에 들어옵니다. 회의하며 함께 그리는 화이트보드가 됩니다.

  • 여러 명이 동시에 편집 — 충돌 없이 실시간 동기화
  • 참여자의 커서와 이름이 실시간으로 표시
  • 연결이 끊겨도 오프라인으로 계속 작업, 다시 연결되면 동기화
  • 내 보드 파일은 내 컴퓨터에 그대로 — 공유 보드에서 나가면 원래 보드로 복귀
